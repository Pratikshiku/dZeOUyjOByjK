### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
1. 登录
2. 个人中心
3. 修改密码
4. 年级管理
5. 专业管理
6. 学生管理
7. 教师管理
8. 教材管理
9. 教材资讯管理
10. 教材评价管理（学生评价管理、教师评价管理）
11. 教材评分可视化（教材评分对比、教材评分比率）
12. 成员举报
13. 安全退出
